About

Phy project is a basic language to simplify creation and control of physics engine on three.js project. Phy make bridge from three to physics in worker or directly.
All engine is compact in one file with embed wasm and worker
This is the compilation of all my physics work experience.

Phy provide same command for all physics engine.
But each engine have is own feature.

  1. [link:https://github.com/saharan/OimoPhysics/ OIMO] is only javascript code; is light, stable and speed for simple project.
  2. [link:https://rapier.rs/ RAPIER] is simple, stable and speed for complete project.
  3. [link:https://github.com/kripken/ammo.js AMMO] is wasm port of Bullet engine, is powefull but old. Use for complex game.
  4. [link:https://github.com/fabmax/physx-js-webidl PHYSX] is wasm port by fabmax of Nvidia Physx engine, is the most recent and powerful. It's one of the fastest and have very stable joint for robotics.
  5. [link:https://github.com/BabylonJS/Babylon.js HAVOK] is wasm port of havok engine by babylone team, industry standard used in majority of game.
  6. [link:https://github.com/jrouwe/JoltPhysics.js JOLT] is wasm port of jolt engine, a new engine very promising made by Jrouwe; full of innovative ideas.

Feature

Phy build provide loader manager, auto envmap, advanced controler, real character, particle engine, terrain editor, advanced geometry, instance geometry.
