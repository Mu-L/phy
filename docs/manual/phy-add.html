phy.add()

add object to simulation

Code Example

phy.add({ name:'yo', type:'box', density:1, size:[2,2,2], pos:[0,2,0], rot:[0,0,0] })

Command

phy.add( [param:Object parameters] )

phy.add( [param:Array list] )

const obj = phy.add( [param:Object parameters] )

Parameters

[param:String name] — name of object is automatic if not set
[param:String type] — type of object
[param:Float density] — is the density of object mass = density * volume
[param:Array size] — size of object
[param:Array pos] — position of object
[param:Array rot] — rotation of object in degrée

Notes

phy.add have a lot of parametre so look editor examples or code in /demos

static object have no mass or density.
for moving object add density or mass.

Type can be


rigid body : 'plane', 'box', 'sphere', 'cylinder', 'capsule', 'convex', 'mesh', 'compound'
extra geometry: 'ChamferBox', 'ChamferCyl', 'highSphere', 'customSphere'
joint constraint : 'prismatic', 'cylindrical', 'generic', 'hinge'

compound is a group of geometry
you can remplace any geomtry by other mesh { mesh:threeObject }
