phy.init()

initialise physics engine

Code Example

import { phy } from 'phy-engine'; phy.init({ type:'OIMO', worker:true, compact:true, callback:startDemo, scene:scene, renderer:renderer, })

Command

phy.init( [param:Object parameters] )

Parameters

[param:String type] — the physics engine type can be 'OIMO', 'AMMO', 'RAPIER', 'JOLT', 'HAVOK', 'PHYSX'
[param:Boolean worker] — run in worker or not ( worker free resource for three )
[param:Boolean compact] — use the compact lzma version of engine in /compact folder
[param:Function callback] — function start after physics engine ready to use
[param:Object3d scene] — the three.js main scene
[param:WebGLRenderer renderer] — the three.js main renderer
[param:String path] — set default resource path

Optional

[param:String envmap] — set envmap url : hdr, exr or hdrjpg
[param:Boolean usePmrem] — use pmremGenerator
[param:Boolean useBackground] — use envmap scene background default = true
